# FIRM Linux Loader

A bootloader for running Linux on the Nintendo 3DS family of devices.

## How to run this

See [the wiki](https://github.com/linux-3ds/firm_linux_loader/wiki). Refer to
"3DS Hacks Guide" for running homebrew software on your device.
